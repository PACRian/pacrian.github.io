
## File tree
To organize the pages, this project looks like this:
```
/
├── _config.yml              # Jekyll configuration
├── index.md                 # Home page
├── blogs.md                 # Blog listing page
├── about.md                 # About page
├── _layouts/                # Custom layouts
│   └── post.html            # Layout for tutorials/posts
├── _tutorials/              # Your tutorial markdown files
│   ├── typedef-tutorial.md
│   ├── another-tutorial.md
│   └── ...
└── assets/                  # CSS, images, etc.
    └── css/
        └── styles.css
        └── ...
```
